# ML-IDS: Hybrid Network Security & Productivity System

**ML-IDS** — это интеллектуальная система мониторинга сетевой активности, объединяющая в себе функции обнаружения вторжений (HIDS) и анализа продуктивности сотрудников.

## 🚀 Основные возможности

- **Security (DDoS Detection):** Эвристический анализ трафика в реальном времени. Мгновенное обнаружение DDoS-атак по всплеску PPS (Packets Per Second).
- **ML Anomaly Detection:** Использование моделей Isolation Forest и Autoencoder (TensorFlow) для выявления скрытых угроз (сканирование портов, эксфильтрация данных).
- **Productivity Tracking:** Автоматический учет времени, проведенного пользователями на различных интернет-ресурсах (YouTube, соцсети и т.д.).
- **Background Worker:** Полностью автономный фоновый процесс, который запускается автоматически и работает в скрытом режиме.
- **Admin Dashboard:** Современный веб-интерфейс с графиками (Chart.js), статистикой трафика и управлением ML-моделями.

## 🛠 Технологический стек

- **Backend:** Python 3.9+, Flask, Flask-SQLAlchemy, Flask-Login.
- **Network Engine:** Scapy + Npcap (DPI анализ).
- **Machine Learning:** TensorFlow (Keras), Scikit-learn, Joblib.
- **Database:** SQLite (WAL Mode для высокой производительности).
- **Frontend:** Bootstrap 5, FontAwesome 6, Chart.js.

## 📋 Архитектура

Система состоит из двух ключевых компонентов:
1. **Web App (`app.py`):** Управляет пользователями, отображает статистику и предоставляет API для управления.
2. **Traffic Worker (`worker.py`):** Фоновый агент. Выполняет захват пакетов, извлечение признаков (13 параметров), прогон через ML-модели и расчет времени сессий.

## ⚙️ Установка и запуск

1. Установите [Npcap](https://npcap.com/#download) в режиме "WinPcap API-compatible".
2. Клонируйте репозиторий.
3. Создайте виртуальное окружение: `python -m venv venv`.
4. Установите зависимости: `pip install -r requirements.txt`.
5. Запустите веб-сервер: `python app.py`.
6. Зайдите на `http://127.0.0.1:5000`, зарегистрируйтесь и создайте свою первую модель.
